# Brand Setup Instructions

## Logo Setup
1. Convert `Logo.pdf` to PNG format:
   - Use Adobe Illustrator, InDesign, or an online PDF to PNG converter
   - Export as `logo.png` in the root directory
   - Recommended size: 200px height (width will scale automatically)
   - Ensure transparent background if possible

## Brand Colors Setup
1. Open `Brand.pdf` and identify the brand colors
2. Update the color variables in `styles.css` (lines 12-34):
   - `--primary-color`: Main brand color (hex code)
   - `--secondary-color`: Secondary brand color (hex code)
   - `--dark-color`: Dark text/background color
   - `--text-dark`: Primary text color
   - `--text-light`: Secondary text color
   - `--light-color`: Light background color
   - `--gradient`: Update gradient colors to match brand
